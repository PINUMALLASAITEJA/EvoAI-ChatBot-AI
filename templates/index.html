About EVO-AI

EVO-AI is a dynamic chatbot platform built using Flask, MongoDB, and the GPT-Neo model to simulate intelligent responses. With integrated factual query detection and memory-based context, it delivers both informative and conversational replies.

Key Features

  • Context-Aware AI Responses
  • Google SERP API for factual queries
  • MongoDB storage for chat history
  • Login & Register authentication system
  • Responsive UI with smooth animations

System Architecture

The system includes a Flask backend that routes user messages to a decision module. Based on the query type, responses are generated via GPT-Neo or fetched through Google Search. All chats are stored securely in MongoDB.

AI Model Integration

GPT-Neo, an open-source transformer-based model from EleutherAI, is used to generate human-like conversations. It is deployed using PyTorch with DirectML (GPU-supported) for improved performance and inference speed.
